# Python Learning MVP

Minimal interactive Python learning prototype built with Next.js (App Router), TypeScript, Tailwind CSS, and a lightweight CodeMirror editor.

Quick start:

1. Install dependencies

```bash
npm install
```

2. Run dev server

```bash
npm run dev
```

Open http://localhost:3000

Notes: The API endpoint `POST /api/run` executes received Python code using `python3` on the host. This is intended for local prototyping only — do not expose to untrusted users.
